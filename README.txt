This Python script is designed to find the shortest path between nodes in a weighted graph.
 It utilizes Dijkstra's algorithm to efficiently determine the shortest path from a starting
 node to a target node.
 The script defines a function shortest_path(graph, start, target='')
 which takes a weighted graph represented as a dictionary, a starting node, and an optional target node.
 It then employs Dijkstra's algorithm to find the shortest path from the starting node to the target node,
 if specified, or to all other nodes in the graph if no target node is specified.
        ------> Dijkstra's Algorithm finds the shortest path between a given node
                (which is called the "source node") and all other nodes in a graph.
                This algorithm uses the weights of the edges to find the path that minimizes the total distance
                (weight) between the source node and all other nodes.

Este script em Python foi desenvolvido para encontrar o caminho mais curto entre pontos em um grifico ponderado.
Ele utiliza o algoritmo de Dijkstra para determinar de forma eficiente o caminho mais curto de um ponto inicial
para um ponto alvo.
O script define uma função shortest_path(graph, start, target='')
que recebe um grafico ponderado representado como um dicionário, um ponto inicial e um ponto alvo opcional.
Ele então utiliza o algoritmo de Dijkstra para encontrar o caminho mais curto do ponto inicial para o ponto alvo,
se especificado, ou para todos os outros pontos no grafico se nenhumponto alvo for especificado.
        ------> O Algoritmo de Dijkstra encontra o caminho mais curto entre um ponto dado
                (chamado de "nó de origem") e todos os outros pontos em um grafico. Este algoritmo utiliza
                os pesos das arestas para encontrar o caminho que minimiza a distância total (peso)
                entre o ponto de origem e todos os outros pontos.
